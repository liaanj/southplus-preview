<h2>South Plus Media Preview</h2>

<p>
  在南+帖子列表页中，自动读取帖子第一页的楼主内容，并在标题下方显示图片和第三方媒体预览。
</p>

<p>
  支持普通图片、附件图片、裸图片链接，以及常见第三方图床链接。对于 Gofile 链接，会自动解析其中的图片和视频；视频会显示封面或占位预览，点击后可在弹窗中播放。
</p>

<h3>主要功能</h3>

<ul>
  <li>在帖子列表页标题下方显示楼主媒体缩略图</li>
  <li>自动展开免费购买获取图片</li>
  <li>支持附件图、外链图床、裸图片 URL</li>
  <li>默认解析第三方媒体链接</li>
  <li>支持 Gofile 图片和视频文件</li>
  <li>Gofile 视频可显示封面，点击后弹窗播放</li>
  <li>横向滚轮浏览缩略图，不影响页面滚动</li>
  <li>自动过滤常见表情、头像等无关图片</li>

</ul>

<h3>使用方式</h3>

<p>
  安装脚本后打开南+帖子列表页即可自动生效。脚本会在后台读取每个帖子的第一页内容，并将可预览媒体显示在标题下方。
</p>

<h3>说明</h3>

<ul>
  <li>Gofile 视频播放需要先通过脚本鉴权加载为本地 Blob，因此大文件可能需要等待下载完成后播放。</li>
  <li>如果 Gofile 视频没有缩略图，会显示视频占位卡片。</li>
</ul>
